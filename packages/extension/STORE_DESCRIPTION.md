# Agentation - Chrome Web Store 등록 정보

## 이름
Agentation

## 짧은 설명 (132자 이내)
Annotate webpage elements and generate structured feedback for AI coding agents like Claude Code, Cursor, and Windsurf.

## 상세 설명
Agentation은 웹페이지 요소를 시각적으로 어노테이션하고, AI 코딩 에이전트를 위한 구조화된 피드백을 생성하는 개발자 도구입니다.

### 주요 기능
• 요소 하이라이트 - 마우스 오버 시 CSS 선택자 표시
• 클릭 어노테이션 - 요소 클릭 후 피드백 작성
• CSS 선택자 자동 추출 - AI가 코드를 찾기 쉬운 정확한 선택자
• 마크다운 출력 - 클립보드에 복사하여 AI에 붙여넣기
• 애니메이션 일시정지 - 특정 프레임 캡처
• 마커 표시/숨김 - 작업 중 마커 토글
• 커스터마이징 - 마커 색상, 출력 상세도 설정

### 사용 방법
1. 우측 하단 아이콘 클릭으로 툴바 펼치기
2. 활성화 버튼 클릭
3. 웹페이지 요소 클릭하여 어노테이션 추가
4. 복사 버튼으로 마크다운 복사
5. Claude Code, Cursor 등 AI 에이전트에 붙여넣기

### AI 에이전트와 함께 사용
Agentation은 코드베이스에 접근할 수 있는 AI 도구(Claude Code, Cursor, Windsurf 등)와 함께 사용할 때 가장 효과적입니다. 
"파란색 버튼"이라고 설명하는 대신, `.sidebar > .nav-actions > button.primary` 같은 정확한 선택자를 제공하여 AI가 코드를 직접 검색할 수 있습니다.

### 개인정보 보호
• 데이터는 로컬에만 저장됩니다
• 외부 서버로 전송되지 않습니다
• 어노테이션은 브라우저 세션 동안만 유지됩니다
